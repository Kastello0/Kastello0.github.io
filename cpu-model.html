Java-Based Processor Simulation

An educational emulator designed to trace the execution cycle of a simple processing unit, modeling register structures, memory access cycles, and assembly instruction execution blocks.

Architecture Simulation Blocks

Interactive Control Dashboard

Java Swing control panel that allows import of custom process workloads via CSV files. Adjust time quantum, and execute.

Multi-Threaded Live Logger

Features an event-logging framework that asynchronously logs clock cycles, incoming process arrivals, and context switches directly onto the UI without interfering with the main rendering thread.

Automated Performance Analytics

Automatically evaluates scheduler efficiency upon simulation completion, calculating and rendering CPU utilization, turnaround time, and latency summaries.

  • Custom Instruction Set Architecture (ISA)
